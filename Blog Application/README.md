# Blogging Application

## Technologies Used

1. ### Database : SQl(Sequelize)
2. ### Backend : NodeJS/Express
3. ### Frontend : HTML5/ CSS3/ JS/ BOOTSTRAP

## Project Structure

```shell

├── index.js              #Server
├── models                #How the data will be stored
│   └── blog.js
├── seed.js               #Dummy dataset
├── db.js                 #Database initialisation
└── views                 #Frontend
    ├── edit.ejs
    ├── index.ejs
    ├── new.ejs
    ├── partials
    │   ├── footer.ejs
    │   ├── header.ejs
    │   └── navbar.ejs
```

## Business Logic

1. Create a new blog
2. Updating the blog
3. Delete a blog
